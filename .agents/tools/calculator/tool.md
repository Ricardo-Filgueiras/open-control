# Ferramenta: Calculadora

Use esta ferramenta para executar operações matemáticas e conversões para o usuário.

## Operações Suportadas:
- **Aritmética básica:** adição, subtração, multiplicação, divisão, potência
- **Porcentagem:** "Quanto é 15% de 200?"
- **Conversões:**
  - Temperatura: Celsius ↔ Fahrenheit ↔ Kelvin
  - Comprimento: metro, km, milha, polegada
  - Peso: kg, g, libra, onça
  - Moeda: conversões cambiais (quando possível)
- **Estatísticas:** média, mediana, desvio padrão
- **Geometria:** área, perímetro, volume

## Exemplos de uso:
- "Quanto é 15% de 200?" → 30
- "Converta 100°F para Celsius" → 37.78°C
- "Qual é a área de um triângulo com base 5 e altura 10?" → 25
- "Qual é a média de 10, 20, 30, 40?" → 25

## Instruções:
1. Identifique o tipo de operação
2. Execute o cálculo com precisão
3. Forneça a resposta final claramente
4. Mostre as etapas para operações complexas
5. Use unidades corretas na resposta

## Formato de resposta:
```
Operação: [descrição]
Cálculo: [etapas]
Resultado: [resposta com unidade]
```
